# ふらっと法務事務所サイト - HTMLコンテンツ構造分析

## flow.html（ご利用の流れページ）の構造分析

### 共通要素
- **ヘッダー部分**
  - メタタグ、タイトル、CSS読み込み
  - サイトロゴ、ナビゲーションメニュー
  - モバイルメニュートグル機能

- **フッター部分**
  - 会社情報（住所、連絡先等）
  - サービスメニュー
  - サイトマップ
  - 関連情報リンク
  - コピーライト

### 個別コンテンツ領域
1. **ページヘッダー**
   ```html
   <section>
     <div>
       <h>ご利用の流れ</h1>
       <p>入札サポートサービスのご利用方法をわかりやすくご説明します</p>
     </div>
   </section>
   ```

2. **利用の流れ概要**
   ```html
   <section>
     <div>
       <h>サービス利用の流れ</h2>
       <div>
         <div>
           <p>ふらっと法務事務所の入札サポートサービスは、お客様のニーズに合わせて柔軟に対応いたします。初めての方でも安心してご利用いただけるよう、わかりやすいステップでサポートいたします。</p>
         </div>
       </div>
     </div>
   </section>
   ```

3. **利用の流れ詳細**
   - 7つのステップで詳細な流れを説明
   - 各ステップには番号、タイトル、説明文、詳細リスト、所要期間などが含まれる
   - ステップ構成:
     1. 無料相談・ヒアリング
     2. 最適プランのご提案
     3. ご契約・サポート開始
     4. 入札参加資格の取得
     5. 入札情報の収集・選定
     6. 入札書類の作成
     7. 入札実施・フォローアップ

4. **よくある質問**
   - アコーディオン形式の質問と回答
   - 4つの質問項目:
     - サービス利用から入札参加までの期間
     - プラン変更の可否
     - 書類代行の範囲
     - 落札できなかった場合のサポート

5. **CTA（Call to Action）セクション**
   ```html
   <section>
     <div>
       <div>
         <div>
           <h>まずは無料相談からお気軽にどうぞ</h2>
           <p>入札に関するお悩みやご質問、お気軽にご相談ください。</p>
           <a>無料相談のお申し込み</a>
         </div>
       </div>
     </div>
   </section>
   ```

### 画像・アセット
- ステップごとの説明画像（7枚）
- アイコン: Font Awesomeを使用

### JavaScript機能
- モバイルメニュートグル
- アコーディオン機能（よくある質問）
- タブ機能
- 画像フォールバック機能

## 移行時の注意点
1. 7つのステップ構造の適切な移行
2. アコーディオン機能の実装
3. 画像パスのJekyll相対URLヘルパーへの変換
4. レスポンシブデザインの維持
